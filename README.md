## A simple Shell with C Language

# Inspired By ALX Africa

# Compilation
- The shell was compiled on Ubuntu 22.04.2 LTS using gcc and its flags as followed:
```ShellSession
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o hsh
```
